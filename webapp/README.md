# ChemiSSUe 홈페이지

## 프로젝트 개요
- **이름**: ChemiSSUe (Chemical Students Society University Experience)
- **목표**: 숭실대학교 화학공학과 소모임을 위한 종합 홈페이지
- **주요 기능**: 게시판 시스템, 일정 관리, 회의록 관리, 활동 사진 갤러리

## 현재 완성된 기능

### 📱 메인 기능
- ✅ **반응형 메인페이지** - 소모임 소개 및 주요 소식 표시
- ✅ **소개 페이지** - ChemiSSUe 활동 및 목표 소개 (지도교수 정보 포함)
- ✅ **9개 카테고리 게시판 시스템** - 완전한 CRUD 기능
- ✅ **사용자 인증 시스템** - 로그인/회원가입/세션 관리
- ✅ **파일 업로드 시스템** - 이미지/문서 첨부 기능
- ✅ **종합 관리자 패널** - 사용자, 권한, 컨텐츠, 통계, 백업 관리
- ✅ **실시간 알림 시스템** - 새 게시글/일정 알림
- ✅ **PWA 기능** - 오프라인 지원 및 앱 설치
- ✅ **강화된 보안 시스템** - 비로그인 사용자 화면 캡처 및 다운로드 방지

### 🗂️ 게시판 카테고리
1. **공지사항** (`/board/notices`) - 중요한 소모임 공지
2. **활동사진** (`/board/photos`) - 다양한 활동 사진 갤러리
3. **일정표** (`/board/schedule`) - 소모임 일정 관리 및 이벤트 캘린더
4. **회의록** (`/board/meetings`) - 회의 내용 및 결정사항 기록
5. **게시판** (`/board/board`) - 자유 소통 공간
6. **건의사항** (`/board/suggestions`) - 소모임 개선 아이디어 제안
7. **홍보자료** (`/board/promotion`) - ChemiSSUe 홍보 및 소개 자료
8. **마음의 편지** (`/board/letters`) - 따뜻한 메시지 전달
9. **자료실** (`/board/resources`) - 학습 및 활동 자료 공유

### 🛠️ 관리자 패널 기능

**사용자 관리**
- ✅ **회원 관리** (`/admin/users`) - 전체 회원 목록, 역할 변경, 계정 관리
- ✅ **가입 승인** (`/admin/users/pending`) - 신규 회원 승인/거부 시스템
- ✅ **권한 관리** (`/admin/permissions`) - 게시판별 세부 권한 설정

**컨텐츠 관리**
- ✅ **페이지 편집** (`/admin/pages`) - 메인/소개/교수소개 페이지 실시간 편집
- ✅ **게시판 관리** - 전체 게시글 및 댓글 관리

**통계 & 분석**
- ✅ **방문 통계** (`/admin/analytics`) - 회원 가입/활동 통계, 인기 게시글 분석
- ✅ **활동 보고서** (`/admin/reports`) - 게시판별 활동 현황, 활발한 회원 분석

**시스템 관리**
- ✅ **백업 관리** (`/admin/backup`) - 데이터베이스 SQL 덤프, CSV 내보내기
- ✅ **메시지 관리** (`/admin/messages`) - 회원 간 메시지 관리 및 통계
- ✅ **공지 발송** (`/admin/broadcast`) - 전체/역할별/개별 알림 발송 시스템

### 🔧 API 엔드포인트

**게시판 API**
- `GET /api/posts?category={slug}&page={num}` - 카테고리별 게시글 목록
- `GET /api/posts/{id}` - 개별 게시글 상세 조회
- `POST /api/posts` - 새 게시글 작성 (인증 필요)
- `GET /api/events` - 일정/이벤트 목록
- `GET /api/meetings` - 회의록 목록

**인증 API**
- `POST /api/auth/login` - 로그인
- `POST /api/auth/register` - 회원가입
- `POST /api/auth/logout` - 로그아웃

**파일 업로드 API**
- `POST /api/upload` - 파일 업로드 (인증 필요)

**알림 API**
- `GET /api/notifications` - 사용자 알림 목록 (인증 필요)
- `POST /api/notifications/{id}/read` - 알림 읽음 처리

**관리자 API**
- `GET /api/admin/users` - 회원 목록 관리 (관리자 전용)
- `POST /api/admin/users/{id}/approve` - 회원 승인/거부
- `POST /api/admin/permissions` - 게시판 권한 설정
- `POST /api/admin/backup/sql` - 전체 데이터베이스 SQL 덤프
- `POST /api/admin/backup/csv/{table}` - 테이블별 CSV 내보내기
- `POST /api/admin/broadcast/notification` - 알림 발송
- `POST /api/admin/broadcast/announcement` - 공고 작성

### 💾 데이터 아키텍처
- **데이터베이스**: Cloudflare D1 SQLite
- **스토리지 서비스**: 
  - D1 Database (게시글, 댓글, 일정, 회의록, 사용자, 파일, 알림)
  - 향후 R2 Storage 연동 가능 (실제 파일 저장)
- **주요 테이블**:
  - `users` - 사용자 관리 (인증, 권한, 프로필, 승인 시스템)
  - `sessions` - 로그인 세션 관리
  - `boards` - 게시판 카테고리 및 설정
  - `board_permissions` - 게시판별 세부 권한 관리
  - `posts` - 게시글 (제목, 내용, 작성자, 조회수, 좋아요)
  - `comments` - 댓글 시스템
  - `events` - 일정 관리
  - `attachments` - 첨부파일
  - `uploaded_files` - 파일 업로드 메타데이터
  - `notifications` - 시스템 알림
  - `messages` - 회원 간 메시지
  - `announcements` - 공고/공지사항
  - `notification_recipients` - 알림 수신자 관리
  - `page_contents` - 페이지 컨텐츠 관리
  - `likes` - 게시글 좋아요
